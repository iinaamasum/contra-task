## Run Locally

Clone the project

```bash
  git clone https://github.com/iinaamasum/contra-task.git
```

Go to the project directory

```bash
  cd contra-task
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```
